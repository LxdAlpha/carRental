updateDateTimeOffset (SQLServerResultSet)

Microsoft SQL Server JDBC Driver 3.0 中新增了此方法。

将指定的列值更新为 DateTimeOffset 类 值。

重载列表

名称

说明

updateDateTimeOffset(int, microsoft.sql.DateTimeOffset)

根据给定的基于零的列序号，将指定的列值更新到 DateTimeOffset 类 值。

updateDateTimeOffset(string, microsoft.sql.DateTimeOffset)

根据给定的列名称，将指定的列值更新到 DateTimeOffset 类 值。
