valueOf 方法 (java.sql.Timestamp, java.util.Calendar)

创建一个 DateTimeOffset 对象，该对象表示按照距 GMT（给定 java.sql.Timestamp 值以及指示偏移量的 java.util.Calendar 值）的特定偏移量的时间点。

语法

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, java.util.Calendar calendar)

参数

timestamp

一个java.sql.Timestamp 值。

calendar

偏移量值。将根据 timestamp 值设置 calendar 的日期和时间组件。

返回值

返回一个 DateTimeOffset 对象，该对象表示按照给定 java.util.Calendar 对象的时区由 java.sql.Timestamp 对象给出的时间点。

注释

此方法还将 java.util.Calendar 对象设置为 java.sql.Timestamp 对象给出的时间点。
