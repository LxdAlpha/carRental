getPortNumber 方法 (SQLServerDataSource)

返回用于与 SQL Server 通信的当前端口号。

语法

public int getPortNumber()

返回值

包含当前端口号的 int 值。

注释

该端口号是打开与 SQL Server 的套接字连接时使用的 TCP/IP 端口号。如果未设置 portNumber 属性，则 getPortNumber 方法将返回默认值 1433。

setPortNumber 方法不对传入的端口值进行任何范围检查。您可以传递无效的端口号（如 99999）而不引发错误。
