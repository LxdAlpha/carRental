getParameterClassName

检索指定 Java 类的完全限定名称，而该 Java 类的实例应传递给 SQLServerPreparedStatement 类中的 setObject 方法。

检索指定 SQLServerPreparedStatement 对象中的参数数目，此 SQLServerParameterMetaData 对象包含该指定对象的信息。
