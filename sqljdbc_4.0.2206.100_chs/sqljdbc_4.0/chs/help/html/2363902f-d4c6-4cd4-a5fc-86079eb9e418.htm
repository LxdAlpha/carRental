方法

名称

说明

addBatch

为此 Statement 对象的一批命令添加一组参数。

cancel

（继承自 SQLServerStatement。）取消当前正由此 Statement 对象运行的 SQL 语句。

clearBatch

为此 SQLServerStatement 对象清除当前 SQL 命令列表。

clearParameters

立即清除当前参数值。

clearWarnings

（继承自 SQLServerStatement。）清除针对此 Statement 对象报告的所有警告。

close

立即释放此 Statement 对象的数据库和 JDBC 资源，而非等待它们自动释放。

execute

运行此 Statement 对象中的 SQL 语句，该语句可以是任何类型的 SQL 语句。

executeBatch

向数据库提交要运行的一批命令。如果所有命令都成功运行，则返回一个更新计数数组。

executeQuery

运行此 Statement 对象中的 SQL 查询并返回由该查询生成的 SQLServerResultSet 对象。

executeUpdate

运行此 Statement 对象中的 SQL 语句，该语句必须是 SQL INSERT、UPDATE、MERGE 或 DELETE 语句；或是不返回任何内容的 SQL 语句，如 DDL 语句。

getConnection

（继承自 SQLServerStatement。）检索 SQLServerConnection 对象，该对象生成了此 Statement 对象。

getFetchDirection

（继承自 SQLServerStatement。）检索从数据库表中提取行的方向，该方向是由此 Statement 对象生成的结果集的默认值。

getFetchSize

（继承自 SQLServerStatement。）检索结果集行数，该行数是由此 Statement 对象生成的结果集对象的默认提取大小。

getGeneratedKeys

（继承自 SQLServerStatement。）检索因运行此 Statement 对象而创建的任何自动生成的键。

getMaxFieldSize

（继承自 SQLServerStatement。）检索可为 SQLServerResultSet 对象（该对象由此 Statement 对象生成）中的字符和二进制列值返回的最大字节数。

getMaxRows

（继承自 SQLServerStatement。）检索由此 Statement 对象生成的 SQLServerResultSet 对象可包含的最大行数。

getMetaData

检索一个 SQLServerResultSetMetaData 类 对象，此对象包含执行 Statement 对象时将返回的 SQLServerResultSet 对象的列的相关信息。

getMoreResults

（继承自 SQLServerStatement。）移动到此 Statement 对象的下一个结果。

getParameterMetaData

检索此 Statement 对象的参数数目、参数类型和参数属性。

getResponseBuffering

（继承自 SQLServerStatement。）检索此 SQLServerStatement 对象的响应缓冲模式。

getQueryTimeout

（继承自 SQLServerStatement。）检索 Microsoft JDBC Driver for SQL Server 将等待此 Statement 对象运行的秒数。

getResultSet

（继承自 SQLServerStatement。）检索当前结果作为 SQLServerResultSet 对象。

getResultSetConcurrency

（继承自 SQLServerStatement。）检索 SQLServerResultSet 对象的结果集并发机制，这些对象由此 Statement 对象生成。

getResultSetHoldability

（继承自 SQLServerStatement。）检索 SQLServerResultSet 对象的结果集保持能力，这些对象由此 Statement 对象生成。

getResultSetType

（继承自 SQLServerStatement。）检索 SQLServerResultSet 对象的结果集类型，这些对象由此 Statement 对象生成。

getUpdateCount

（继承自 SQLServerStatement，）检索当前结果作为更新计数。

getWarnings

（继承自 SQLServerStatement。）检索调用此 Statement 对象时报告的第一个警告。

isClosed

（继承自 SQLServerStatement。）指示是否已关闭此 Statement 对象。

isPoolable

（继承自 SQLServerStatement。）返回指示是否可以将语句添加到用户提供的语句池的一个值。

isWrapperFor

指示此语句对象是否为指定接口的包装。
