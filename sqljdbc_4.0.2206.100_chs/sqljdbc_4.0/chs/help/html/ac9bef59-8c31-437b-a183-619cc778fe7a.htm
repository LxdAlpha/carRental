last 方法 (ISQLServerResultSet)

将游标移到此 SQLServerResultSet 对象中的最后一行。

语法

public boolean last()

返回值

如果新的当前行有效，则为 true。如果没有要处理的其他行，则为 false。

异常

SQLServerException

注释

此 last 方法由 java.sql.ResultSet 接口中的 last 方法指定。
