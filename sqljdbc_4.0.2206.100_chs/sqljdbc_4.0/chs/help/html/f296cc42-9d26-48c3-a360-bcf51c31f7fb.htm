insertsAreDetected 方法 (SQLServerDatabaseMetaData)

检索是否可通过调用 SQLServerResultSet 类的 rowInserted 方法检测可见行插入。

语法

public boolean insertsAreDetected(int type)

参数

type

指示结果集类型的整数，它可以为 java.sql.ResultSet 或 SQLServerResultSet 中定义的以下值之一：

java.sql.ResultSet 类型

TYPE_FORWARD_ONLY

TYPE_SCROLL_SENSITIVE

TYPE_SCROLL_INSENSITIVE

SQLServerResultSet 类型

TYPE_SS_SCROLL_STATIC

TYPE_SS_SCROLL_KEYSET

TYPE_SS_DIRECT_FORWARD_ONLY

TYPE_SS_SERVER_CURSOR_FORWARD_ONLY

TYPE_SS_SCROLL_DYNAMIC

返回值

如果可以检测行插入，则为 true。否则为 false。

异常

SQLServerException

注释

此 insertsAreDetected 方法由 java.sql.DatabaseMetaData 接口中的 insertsAreDetected 方法指定。

SQL Server 不检测任何游标类型的插入行。
