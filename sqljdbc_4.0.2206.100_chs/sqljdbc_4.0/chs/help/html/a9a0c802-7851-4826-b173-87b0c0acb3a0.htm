setObject 方法 (java.lang.String, java.lang.Object, int)

使用给定的对象和目标类型设置指定参数的值。

语法

public void setObject(java.lang.String sCol,
java.lang.Object o,
int n)

参数

sCol

包含参数名称的 String。

o

Object 值。

n

指示 java.sql.Types 中定义的目标类型的 int 值。

异常

SQLServerException

注释

此 setObject 方法由 java.sql.CallableStatement 接口中的 setObject 方法指定。

从 SQL Server JDBC Driver 3.0 开始，此方法的行为由 sendTimeAsDatetime 连接属性（设置连接属性）和 SQLServerDataSource.setSendTimeAsDatetime 修改。

有关详细信息，请参阅配置如何将 java.sql.Time 值发送到服务器。
