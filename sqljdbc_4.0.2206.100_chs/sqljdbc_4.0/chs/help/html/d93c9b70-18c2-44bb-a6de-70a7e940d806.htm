getParameterMode 方法 (SQLServerParameterMetaData)

检索指定参数的模式。

语法

public int getParameterMode(int param)

参数

param

指示参数索引的 int 值。

返回值

指示指定参数的模式的 int 值，可以为以下值之一：

ParameterMetaData.parameterModeIn

ParameterMetaData.parameterModeInOut

ParameterMetaData.parameterModeOut

ParameterMetaData.parameterModeUnknown

异常

SQLServerException

注释

此 getParameterMode 方法由 java.sql.DatabaseMetaData 接口中的 getParameterMode 方法指定。
