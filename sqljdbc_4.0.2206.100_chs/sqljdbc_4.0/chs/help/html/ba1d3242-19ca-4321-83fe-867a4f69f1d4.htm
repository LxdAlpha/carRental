ISQLServerDataSource 接口

一个工厂，用于创建与此对象表示的数据源的连接。SQL Server JDBC Driver 3.0 中新增了此接口。

包：com.microsoft.sqlserver.jdbc

扩展：java.sql.CommonDataSource

语法

public interface ISQLServerDataSource

注释

此接口由 SQLServerDataSource 类 实现。

此接口公开以下 Microsoft JDBC Driver for SQL Server 特定的方法：
