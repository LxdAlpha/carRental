getUnicodeStream 方法 (int)

检索此 SQLServerResultSet 对象的当前行中指定列索引的值作为 Unicode 字符流。

此方法在 JDBC 规范中已不推荐使用，调用它将导致引发“尚未实现”异常。您应改用 getCharacterStream 方法。

语法

public java.io.InputStream getUnicodeStream(int columnIndex)

参数

columnIndex

指示列索引的 int 值。

返回值

InputStream 对象。

异常

SQLServerException

注释

此 getUnicodeString 方法由 java.sql.ResultSet 接口中的 getUnicodeString 方法指定。
