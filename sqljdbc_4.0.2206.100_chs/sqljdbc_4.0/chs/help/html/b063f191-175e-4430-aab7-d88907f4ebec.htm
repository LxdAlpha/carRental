说明

free

此方法释放 NCLOB 对象以及它所持有的资源。

getAsciiStream

检索由 java.sql.NClob 对象指定的 NCLOB 值作为 ASCII 流。

getCharacterStream

检索由 java.sql.NClob 对象指定的 NCLOB 值。

getSubString

在由 java.sql.NClob 对象指定的 NCLOB 值中检索指定子字符串的副本。

length

在由 java.sql.NClob 对象指定的 NCLOB 值中检索字符数。

position

检索指定 java.sql.NClob 对象的字符位置或根据指定的起始位置检索 java.sql.NClob 中的子字符串。

setAsciiStream

检索用来在指定起始位置将 ASCII 字符写入此 java.sql.NClob 对象表示的 NCLOB 值的流。

setCharacterStream

检索用来在指定起始位置将 Unicode 字符流写入此 java.sql.NClob 对象表示的 NCLOB 值的流。

setString

将给定的 String 写入 NCLOB（从指定位置开始）。

truncate

将 NCLOB 值截断为指定长度。
