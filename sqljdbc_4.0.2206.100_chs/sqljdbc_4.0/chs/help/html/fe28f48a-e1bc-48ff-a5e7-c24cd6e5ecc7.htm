使用 SQL 语句

使用 Microsoft JDBC Driver for SQL Server 和内联 SQL 语句处理 SQL Server 数据库中的数据时，有多个不同的类可供使用。使用哪个类取决于要运行的 SQL 语句的类型。

如果 SQL 语句不包含 IN 参数，则使用 SQLServerStatement 类；但如果包含 IN 参数，则使用 SQLServerPreparedStatement 类。

如果需要使用同时包含 IN 和 OUT 参数的 SQL 语句，则必须将这些语句作为存储过程来实现，并使用 SQLServerCallableStatement 类对其进行调用。有关使用存储过程的详细信息，请参阅使用带有存储过程的语句。
