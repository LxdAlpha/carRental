execute 方法 (java.lang.String, java.lang.String[])

运行可返回多项结果的给定的 SQL 语句，并通知 Microsoft JDBC Driver for SQL Server 以指明给定数组中指示的自动生成的键应对检索可用。

语法

public final boolean execute(java.lang.String sql,
                             java.lang.String[] columnNames)

参数

sql

包含 SQL 语句的 String。

columnNames

一个字符串数组，指示哪些自动生成的键的列名应可用。

返回值

如果第一个结果为一个结果集，则为 true。否则为 false。

异常

SQLServerException

注释

此 execute 方法由 java.sql.Statement 接口中的 execute 方法指定。
