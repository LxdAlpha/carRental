setSendTimeAsDatetime 方法 (SQLServerDataSource)

SQL Server JDBC Driver 3.0 中新增了此方法。

修改 sendTimeAsDatetime 连接属性的设置。

语法

public void setSendTimeAsDatetime(boolean sendTimeAsDateTime)

参数

sendTimeAsDateTime

布尔值。如果设置为 true，java.sql.Time 值将作为 SQL Serverdatetime 类型发送到服务器。如果设置为 false，java.sql.Time 值将作为 SQL Servertime 类型发送到服务器。

注释

SQLServerDataSource.getSendTimeAsDatetime 返回 sendTimeAsDatetime 连接属性的设置。

有关 sendTimeAsDatetime 连接属性的详细信息，请参阅设置连接属性。

有关详细信息，请参阅配置如何将 java.sql.Time 值发送到服务器。
