setFetchSize 方法 (SQLServerStatement)

为 Microsoft JDBC Driver for SQL Server 提供提示以指明在需要更多行时应从数据库中提取的行数。

语法

public final void setFetchSize(int rows)

参数

rows

一个指示要提取的行数的 int。

异常

SQLServerException

注释

此 setFetchSize 方法由 java.sql.Statement 接口中的 setFetchSize 方法指定。
