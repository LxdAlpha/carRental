getColumnTypeName 方法 (SQLServerResultSetMetaData)

检索指定列的数据库特定类型名称。

语法

public java.lang.String getColumnTypeName(int column)

参数

column

指示列索引的 int。

返回值

包含列的服务器名称的 String。

异常

SQLServerException

注释

此 getColumnTypeName 方法由 java.sql.ResultSetMetaData 接口中的 getColumnTypeName 方法指定。

Microsoft SQL Server JDBC Driver 3.0 在 TYPE_NAME 列中行为发生更改。有关详细信息，请参阅 SQLServerDatabaseMetaData.getColumns。
