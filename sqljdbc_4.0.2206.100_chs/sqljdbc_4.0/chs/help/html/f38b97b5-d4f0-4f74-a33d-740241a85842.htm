setString 方法 (SQLServerCallableStatement)

将指定参数设置为给定的 Java String 值。

语法

public void setString(java.lang.String sCol,
                      java.lang.String s)

参数

sCol

包含参数名称的 String。

s

String 值。

异常

SQLServerException

注释

此 setString 方法由 java.sql.CallableStatement 接口中的 setString 方法指定。

仅在 Microsoft JDBC Driver for SQL Server 知道目标类型为二进制时，方会执行字符串到二进制的转换。如果 JDBC 驱动程序不知道基础类型，它将会传递 String 文本并在服务器不能执行转换时返回一个服务器错误。
