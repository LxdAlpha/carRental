了解 JDBC 驱动程序数据类型

Microsoft JDBC Driver for SQL Server 支持在使用 SQL Server 作为数据库的 Java 应用程序中使用 JDBC 的基本数据类型和高级数据类型。

JDBC 类型系统负责调解 SQL Server 数据类型与 Java 语言类型和对象之间的转换。JDBC 类型的模型建立在 SQL-92 和 SQL-99 类型的基础之上。JDBC 驱动程序符合 JDBC 规范，且设计为在可预测性与灵活性之间提供适当的平衡。

这部分的各个主题介绍如何使用基本数据类型和高级数据类型，以及如何可以将数据类型转换为其他数据类型。
