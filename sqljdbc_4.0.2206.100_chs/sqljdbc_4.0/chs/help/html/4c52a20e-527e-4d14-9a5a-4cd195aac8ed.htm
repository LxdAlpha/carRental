executeUpdate 方法 (java.lang.String, int)

运行给定 SQL 语句并向 Microsoft JDBC Driver for SQL Server 发送有关由此 SQLServerStatement 对象自动生成的键是否应对检索可用的给定标志。

语法

public final int executeUpdate(java.lang.String sql,
                               int flag)

参数

sql

包含 SQL 语句的 String。

flag

指示是否应使用自动生成的键的 int 值。它必须是下列常量之一：

RETURN_GENERATED_KEYS

NO_GENERATED_KEYS

返回值

一个指示受影响的行数的 int，如果使用 DDL 语句，则为 0。

异常

SQLServerException

注释

此 executeUpdate 方法由 java.sql.Statement 接口中的 executeUpdate 方法指定。

如果执行存储过程将产生大于 1 的更新计数，或生成多个结果集，则请使用 execute 方法执行存储过程。
