SQLServerBlob 构造函数 (SQLServerConnection, byte[])

在给定 SQLServerConnection 对象和 byte 数组时，初始化 SQLServerBlob 类的新实例。

JDBC 驱动程序 2.0 版已不推荐使用此方法，而是使用 SQLServerConnection 类的 createBlob 方法。

语法

public SQLServerBlob(SQLServerConnection connection,
byte[] data)

参数

connection

SQLServerConnection 对象。

data

byte 数组。
