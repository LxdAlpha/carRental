SQLServerStatement 类

表示 JDBC 语句功能的基本实现。

包：com.microsoft.sqlserver.jdbc

实现：ISQLServerStatement

语法

public class SQLServerStatement

注释

SQLServerStatement 类还提供了用于 JDBC 预定义语句和可调用语句的若干基类实现方法。SQLServerStatement 类的基本作用是运行 SQL 语句，然后向用户应用程序返回更新计数和结果集。

此类支持取消对 SQLServerStatement 类、ISQLServerStatement 接口和 java.sql.Statement 接口的包装。有关详细信息，请参阅包装和接口。
