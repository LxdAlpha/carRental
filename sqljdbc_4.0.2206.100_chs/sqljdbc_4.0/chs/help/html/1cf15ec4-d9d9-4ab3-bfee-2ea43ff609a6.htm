setSavepoint 方法 (java.lang.String)

在当前事务中创建具有给定名称的保存点，然后返回表示此保存点的新 SQLServerSavepoint 对象。

语法

public java.sql.Savepoint setSavepoint(java.lang.String sName)

参数

sName

包含保存点名称的 String 值。

返回值

SavePoint 对象。

异常

SQLServerException

注释

此 setSavePoint 方法由 java.sql.Connection 接口中的 setSavePoint 方法指定。

Microsoft JDBC Driver for SQL Server 会自动转义 sName 参数。
