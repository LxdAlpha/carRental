getDate 方法 (java.lang.String, java.util.Calendar)

根据给定的参数名称和 Calendar 对象，检索指定参数值作为 Java 编程语言中的 java.sql.Date 对象。

语法

public java.sql.Date getDate(java.lang.String sCol,
java.util.Calendar cal)

参数

sCol

包含参数名称的 String。

cal

Calendar 对象。

返回值

Date 对象。

异常

SQLServerException

注释

此 getDate 方法由 java.sql.CallableStatement 接口中的 getDate 方法指定。

此方法返回 SQL Server datetime 或 smalldatetime 数据类型的有效日期部分，时间部分设置为 Java 时间基准 00:00（午夜）。
