getBytes 方法 (SQLServerCallableStatement)

检索指定参数的值作为字节数组。

重载列表

名称

说明

getBytes (int)

根据给定的参数索引，检索指定参数的值作为字节值数组。

getBytes (java.lang.String)

根据给定的参数名称，检索指定参数的值作为字节值数组。

备注

在之前的 Microsoft JDBC Driver for SQL Server 版本中，您可使用 SQLServerCallableStatement.getBytes 将字节数组和 SQL Server 数据类型 date、time、datetime2 或 datetimeoffset 的值相互转换。现在对于这些数据类型使用此方法将导致异常，指出不支持该转换。
