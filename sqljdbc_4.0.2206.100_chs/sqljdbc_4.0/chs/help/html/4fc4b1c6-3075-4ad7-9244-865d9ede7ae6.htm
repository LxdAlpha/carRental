使用结果集

使用 SQL Server 数据库中包含的数据时，有一种数据处理方法是使用结果集。Microsoft JDBC Driver for SQL Server 支持通过 SQLServerResultSet 对象使用结果集。通过使用 SQLServerResultSet 对象，您可以检索 SQL 语句或存储过程返回的数据，并根据需要更新数据，然后将该数据保存回数据库中。 

此外，SQLServerResultSet 对象还提供了多种方法，可用于在其数据行中进行导航、获取或设置其中包含的数据，以及建立针对基础数据库中更改的各种敏感度级别。

有关管理结果集的详细信息（包括其更改敏感度），请参阅通过 JDBC 驱动程序管理结果集。

此部分的主题说明了使用结果集来处理 SQL Server 数据库中所包含数据的各种方法。
