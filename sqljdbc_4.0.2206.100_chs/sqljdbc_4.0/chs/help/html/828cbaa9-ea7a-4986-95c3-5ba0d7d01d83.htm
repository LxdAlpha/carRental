继承的字段

java.sql.Statement

CLOSE_ALL_RESULTS、CLOSE_CURRENT_RESULT、EXECUTE_FAILED、KEEP_CURRENT_RESULT、NO_GENERATED_KEYS、RETURN_GENERATED_KEYS、SUCCESS_NO_INFO

方法

名称

说明

addBatch

将给定的 SQL 命令添加至此 SQLServerStatement 对象的当前命令列表。

cancel

取消当前正由此 SQLServerStatement 对象运行的 SQL 语句。

clearBatch

为此 SQLServerStatement 对象清除当前 SQL 命令列表。

clearWarnings

清除对此 SQLServerStatement 对象报告的所有警告。

close

立即释放此 SQLServerStatement 对象的数据库和 JDBC 资源，而非等待它们自动释放。

execute

运行可返回多个结果的给定的 SQL 语句。

executeBatch

向数据库提交要运行的一批命令。如果所有命令都成功运行，则返回一个更新计数数组。

executeQuery

运行给定的 SQL 语句并返回单一的 SQLServerResultSet 对象。

executeUpdate

运行给定的 SQL 语句，可以是 INSERT、UPDATE、MERGE 或 DELETE 语句；或不返回任何内容的 SQL 语句，例如 SQL DDL 语句。

getConnection

检索 SQLServerConnection 对象，该对象生成了此 SQLServerStatement 对象。

getFetchDirection

检索从数据库表中提取行的方向，该方向是由此 SQLServerStatement 对象生成的结果集的默认值。

getFetchSize

检索结果集行数，该行数是由此 SQLServerStatement 对象生成的结果集对象的默认提取大小。

getGeneratedKeys

检索因运行此 SQLServerStatement 对象而创建的任何自动生成的键。

getMaxFieldSize

检索可为由此 SQLServerStatement 对象生成的 SQLServerResultSet 对象中的字符和二进制列值返回的最大字节数。

getMaxRows

检索由此 SQLServerStatement 对象生成的 SQLServerResultSet 对象可包含的最大行数。

getMoreResults

移到此 SQLServerStatement 对象的下一个结果。

getQueryTimeout

检索 Microsoft JDBC Driver for SQL Server 等待此 SQLServerStatement 对象运行的秒数。

getResponseBuffering

检索此 SQLServerStatement 对象的响应缓冲模式。

getResultSet

检索当前结果作为 SQLServerResultSet 对象。

getResultSetConcurrency

检索 SQLServerResultSet 对象的结果集并发机制，这些对象由此 SQLServerStatement 对象生成。

getResultSetHoldability

检索 SQLServerResultSet 对象的结果集保持能力，这些对象由此 SQLServerStatement 对象生成。

getResultSetType

检索 SQLServerResultSet 对象的结果集类型，这些对象由此 SQLServerStatement 对象生成。

getUpdateCount

检索当前结果作为更新计数。

getWarnings

检索调用此 SQLServerStatement 对象时报告的第一个警告。

isClosed

指示此 SQLServerStatement 对象是否已关闭。

isPoolable

返回指示是否可以将语句添加到用户提供的语句池的一个值。

isWrapperFor

指示此语句对象是否为指定接口的包装。

setCursorName

将 SQL 游标名称设置为给定字符串，该字符串将由随后的执行方法使用。

setEscapeProcessing

设置转义处理模式。

setFetchDirection

为 JDBC 驱动程序提供提示以指明处理结果集行时应采用的方向。

setFetchSize

为 JDBC 驱动程序提供提示以指明在需要更多行时应从数据库中提取的行数。

setMaxFieldSize

将存储字符或二进制值的 SQLServerResultSet 列中的最大字节数限制设置为给定的字节数。

setMaxRows

将任何 SQLServerResultSet 对象可包含的最大行数限制设置为给定的数目。

setPoolable

请求语句入池或不入池。

setQueryTimeout

将驱动程序等待 SQLServerStatement 对象运行的秒数设置为给定的秒数。

setResponseBuffering

将此 SQLServerStatement 对象的响应缓冲模式设置为不区分大小写的 String full 或 adaptive。

unwrap

返回一个实现指定接口的对象，从而允许访问特定于 Microsoft JDBC Driver for SQL Server 的方法。
