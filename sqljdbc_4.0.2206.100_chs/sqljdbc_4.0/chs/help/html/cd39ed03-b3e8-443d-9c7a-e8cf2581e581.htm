getShort 方法 (java.lang.String)

根据给定的参数名称，检索指定参数的值作为 Java 编程语言中的 short。

语法

public short getShort(java.lang.String sCol)

参数

sCol

包含参数名称的 String。

返回值

short 值。

异常

SQLServerException

注释

此 getShort 方法由 java.sql.CallableStatement 接口中的 getShort 方法指定。

只有可以安全返回整数值的 SQL Server 数据类型（例如 smallint、tinyint 和 bit）才支持此方法。在任何其他数据类型上使用此方法会引发异常。
