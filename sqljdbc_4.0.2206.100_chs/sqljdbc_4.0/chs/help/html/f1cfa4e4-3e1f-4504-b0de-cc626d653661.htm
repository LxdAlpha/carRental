getNCharacterStream 方法 (int) (SQLServerResultSet)

检索 SQLServerResultSet 对象的当前行中指定列的值作为 Reader 对象。

语法

public java.io.Reader getNCharacterStream(int columnIndex)

参数

columnIndex

指示列索引的 int。

返回值

Reader 对象。

异常

SQLServerException

注释

此 getNCharacterStream 方法由 java.sql.ResultSet 接口中的 getNCharacterStream 方法指定。

此方法可用于检索此 SQLServerResultSet 对象的当前行的 nvarchar、nchar、nvarchar(max)、ntext 或 xml 列的值。如果尝试使用此方法检索其他数据类型的值，则会引发异常。
