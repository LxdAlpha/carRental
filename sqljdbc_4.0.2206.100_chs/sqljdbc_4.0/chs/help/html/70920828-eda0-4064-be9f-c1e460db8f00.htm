setServerName 方法 (SQLServerDataSource)

设置运行 SQL Server 的计算机的名称。

语法

public void setServerName(java.lang.String serverName)

参数

serverName

包含服务器名称的 String。

注释

服务器名称为运行 SQL Server 的目标计算机的主机名。如果未设置 serverName 属性，getServerName 将返回默认值 Null。
