借助 JDBC 驱动程序提高性能和可靠性

所有应用程序开发的一个共同特点是需要不断地提高性能和可靠性。Microsoft JDBC Driver for SQL Server 提供了多种可帮助您实现这一点的技术。

此部分中的主题说明了多种技术，可以在使用 JDBC 驱动程序时用于提高应用程序性能和可靠性。
