getURL 方法 (int)

根据给定的参数索引，检索指定参数的值作为 Java 编程语言中的 URL 对象。

语法

public java.net.URL getURL(int n)

参数

n

指示参数索引的 int 值。

返回值

URL 对象。

异常

SQLServerException

注释

此 getURL 方法由 java.sql.CallableStatement 接口中的 getURL 方法指定。
